🔧 How to Use:

# Start server with default settings (localhost:8080)
python3 server.py

# Start with custom port
python3 server.py 9000

# Start with custom host and port
python3 server.py 9000 0.0.0.0

# Start with custom document root
python3 server.py 8080 127.0.0.1 /path/to/files
